Welcome!

Hi ! Welcome to your personal database.

What is it ?

Here, you can store your thoughts, information, etc., by creating documents, todo-lists, memos...
Whatever you want, in text format.

Let's take an example. Imagine you have a huge library, you need something to store information about books, their place in the shelves, etc. It is possible to use this database to do so. Another example : You are used to writing your thoughts on paper during home-to-work transportation, because your write poems. Now you can forget paper.

How does it work ?

This tool allows you to create an unlimited number of « items ». A book, for instance, is an item.

Each item belongs to a « collection », e.g.: Books.

When your database becomes important, you can even categorize your collections by splitting them into different categories.

Completely offline

The program does not rely on a remote storage system : everything is stored in your web browser !

Once you have loaded this web page (alright if you're reading this), you can use the database and relax because nothing will be send over internet : no external script from untrusted source included, no communication between the webpage and the server. You can even download this HTML web page (Ctrl+S), save it to your disk, and use it as a program that you open in the browser.

Technical information

Storage

The data is stored in your browser's localStorage. The maximum size available then depends on the browser. Normally, it would be 5MB. Note that the data will only be available in the browser you use to save it. If you change your web browser, remember you will lose the data.

To know what is the maximum size of your database with this browser.

Compression

The tool uses gzip to compress the data. This allows you to store more data.

Security

If you're concerned about confidentiality, note that it's possible to choose a password to encrypt the data (settings). Accessing the database will be possible only after having entered the correct password. Note that if you forget the password, you won't be able to retrieve the data.
